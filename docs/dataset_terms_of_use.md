## 1. Dataset Purpose

The Dataset is provided to support non-commercial research on object dynamics.

The Dataset was collected to capture `bowling` and `cube_stacks` scenarios. Although humans
may be incidentally visible in recordings, the Dataset is not intended for
human-centered analysis.
